Overview

  • Plots a trailing line using ATR distance from price.
  • Useful as a visual stop guide or review overlay.
  • Designed to stay unobtrusive on dark/light themes.

Quick Start

  1. Attach to any chart and timeframe.
  2. Set ATRPeriod and ATRMultiplier for your desired distance.
  3. Choose UpdateOnClose if you prefer updates once per bar.
  4. Adjust line color/width to your chart style.

Main Inputs (summary)

  • ATRPeriod — lookback for ATR.
  • ATRMultiplier — distance factor from price.
  • UpdateOnClose — true = update only at candle close.
  • LineColor / LineWidth / Style — visual settings.

Notes

  • No signals or claims; a visual tool only.
