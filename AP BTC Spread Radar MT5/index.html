AP BTC Spread Radar MT5

Real-time spread monitor (points) with an optional moving-average baseline. Lightweight, BTC-friendly. Support via MQL5 messages/comments only.

Get on MQL5 Quick Start Pinned Comment

What it does

  • Displays the **live spread** in points for the current symbol.
  • Optionally plots a **moving-average** of the spread for context (quiet vs. volatile periods).
  • HUD comment (optional) for quick “spread: X pts | MA(N): Y pts”.
  • Very low resource usage — safe to leave on BTC M5/M1.

How to use

  1. Drop the indicator on **BTCUSD** (or any symbol you want to check).
  2. Choose the **MA period** for the spread baseline (or set to 0 to hide).
  3. Optionally enable the **chart comment** and set how often it updates.
  4. If you see unusually high spread compared to the MA, consider pausing new entries.

Tip: On brokers with 10k points ≈ $1 for BTC, “100 pts” ≈ 1¢. Adjust your intuition to your broker’s point scale.
