Daily Open Line

Draws today’s open price as a clean reference line. Optional prior-day open, labels, and neutral styling.

Get on MQL5 ⬅ Back to Home

What it shows

  • Today’s open price as a horizontal line.
  • Optional prior-day open (and a small lookback) for context.
  • Compact label with price (optional), designed for dark/light charts.

Quick Start

  1. Attach to any chart & timeframe.
  2. Toggle ShowTodayOpen and (optionally) ShowPrevOpen.
  3. Set LookbackDays for how many previous opens to draw.
  4. Adjust line color/width and label visibility to match your theme.

Main Inputs (summary)

  • ShowTodayOpen / ShowPrevOpen — toggle current and prior opens.
  • LookbackDays — number of previous opens to draw (if enabled).
  • LineColor / Width / Style — visual settings (kept neutral).
  • ShowLabels / LabelFormat — price label options.

Notes

  • No signals/claims — purely a reference line for planning and reviews.
  • Support via MQL5 messages/comments only.
