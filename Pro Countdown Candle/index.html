Pro Countdown Candle

Minimal time-left display per bar. Honors chart theme and stays lightweight.

Get Pro Countdown Candle (free)

Quick Start

  1. Attach to any chart (works on all TFs).
  2. Pick corner/offset and font size.
  3. Enable seconds if you want finer granularity.

Main Inputs (summary)

  • Corner / X / Y – position on chart.
  • Font / Size – text styling.
  • ShowSeconds – toggle seconds display.
  • ColorAuto – auto-contrast with chart theme.
