ICT Liquidity Sweep (FVG) MT5

Marks equal-high/equal-low “liquidity pools” and alerts when price sweeps them and closes back inside. Optional ATR-based displacement filter. Non-trading, MT5-only.

What it does

  • Scans a lookback window for equal highs and equal lows (“liquidity pools”).
  • Optionally draws horizontal lines at those pools for clear visual context.
  • Marks sweep down when price takes equal highs (EQH) and closes back below.
  • Marks sweep up when price takes equal lows (EQL) and closes back above.
  • Optional ATR/body displacement filter to ignore tiny or weak sweeps.
  • Non-trading: does not open/close orders; it is a visual/alert tool only.

How sweeps are detected (logic)

  • Within InpLookbackBars, the indicator finds bars whose highs/lows are within InpEqualTolPoints of each other → these form EQH/EQL pools.
  • For an EQH sweep:
    • Current bar’s high goes above the EQH price.
    • Current bar’s close comes back below the EQH price.
    • Distance from pool to extreme is at least InpMinSwingGapPoints.
    • If displacement is enabled, candle body must be ≥ InpDisp_MinBodyATR × ATR.
  • For an EQL sweep:
    • Current bar’s low goes below the EQL price.
    • Current bar’s close comes back above the EQL price.
    • Distance from pool to extreme is at least InpMinSwingGapPoints.
    • Same optional ATR/body displacement check.
  • Sweeps are drawn as arrow markers below/above the bar to keep charts clean.

Main Inputs (summary)

  • InpLookbackBars — bars to scan for equal highs/lows.
  • InpEqualTolPoints — tolerance (points) for “equal” highs/lows.
  • InpMinSwingGapPoints — minimum distance from pool to sweep extreme.

Displacement filter

  • InpUseDisplacement — enable/disable body/ATR filter.
  • InpDisp_ATR_Period — ATR period used for filter.
  • InpDisp_MinBodyATR — minimum candle body as a fraction of ATR (e.g. 0.4).

Visuals

  • InpMarkPools — draw pool lines (EQH/EQL) or not.
  • InpShowLines — toggle horizontal pool lines.
  • InpColorEQH / InpColorEQL — colors for EQH/EQL lines.
  • InpColorSweepUp / InpColorSweepDn — arrow colors for sweeps.

Alerts

  • InpAlertsOnSweep — turn sweep alerts on/off.
  • InpUsePopup — terminal popup alerts.
  • InpUseSound + InpSoundFile — sound alerts.
  • InpUsePush — mobile push notifications (if configured).

Usage notes

  • Attach to any BTC/FX symbol and timeframe; logic is timeframe-agnostic.
  • Use modest InpLookbackBars to keep the chart uncluttered.
  • ATR displacement filter helps ignore tiny “noise” sweeps.
  • Combine with your own bias/structure; this is not a standalone trading system.
  • No DLLs. No WebRequest. English-only inputs.
