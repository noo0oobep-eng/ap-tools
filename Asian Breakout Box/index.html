AP Asian Breakout Box (MT5)

Asian session time-box breakout with configurable hours, buffered pendings, and optional trend filter.

Get on MQL5 ⬅ Back to Home

Quick Start

  1. Set your Asian session start/end (server time).
  2. Choose breakout buffer (points) and SL/TP method (ATR or fixed).
  3. Enable trend filter if you want only with-trend trades.
  4. Keep pending expiry to prevent stale orders.

Inputs (summary)

  • SessionStart/End – the time window for the box.
  • BufferPoints – distance beyond high/low for pending orders.
  • ATRPeriod/Mult or SL/TP Points.
  • UseFixedLots / Risk$ for sizing.
