Overview

  • Trails open positions by an ATR-based stop line (per symbol).
  • Optional partial exits at fixed R or distance steps.
  • Daily loss guard (by % or money) to pause management.
  • Works quietly alongside other EAs; magic/symbol filters prevent conflicts.

Quick Start (MT5)

  1. Attach to any chart (one chart can manage multiple symbols).
  2. Set SymbolsToManage (or keep “current symbol” if applicable).
  3. Choose ATRPeriod and ATRMult for the trailing distance.
  4. Enable partials if needed (size and trigger distance).
  5. Keep daily guard at default or set your own %/money cap.

Main Inputs (summary)

  • ATRPeriod / ATRMult – trailing distance.
  • TrailMode – e.g., step/candle-close update style.
  • Partials – enable, step size, and lot sizes.
  • DailyLossGuard% / DailyLossMoney – pause when exceeded.
  • Magic/Symbol filters – manage only the intended positions.

Notes

  • No martingale/grid. Neutral styling and calm order flow.
