AP NY Breakout PRO (MT5)

13:30–14:30 (server) session box breakout via pending orders. SL at opposite side (+extra), TP by R-multiple. One set per window, validator-friendly gates.

Get on MQL5 Quick Start Inputs Preset (.set) Version

What it does

  • Window: 13:30–14:30 (server). Box from InpBoxTF (default M5).
  • Orders: Buy Stop above box high (+buffer) / Sell Stop below box low (–buffer).
  • Exits: SL at opposite side (+extra). TP by R-multiple.
  • Flow: One placement per completed window; pendings auto-expire same day.
  • Safety: OrderCheck, stops-level & margin gates, generous spread default.
  • Options: EMA trend filter, tiny/huge box skip, quick-exit.

How it trades

  1. After the window ends, EA reads high/low of the box (M5 by default).
  2. Places Buy Stop above high (+buffer) and/or Sell Stop below low (–buffer).
  3. SL is the opposite side plus InpSLExtraPoints; TP is by InpTP_R_Multiple.
  4. Only one set is attempted per completed window; pendings expire in InpPendingExpiryHours.
  5. Validator-safe: relies on expiry; opposite-side cancel is OFF by default.
